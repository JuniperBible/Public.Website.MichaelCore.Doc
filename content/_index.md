# Michael Bible Module

Welcome to the Michael Bible Module documentation.

## Overview

Michael is a comprehensive Bible module providing access to biblical texts, translations, and study resources.

## Features

- Multiple Bible translations support
- Verse lookup and search
- Cross-references
- Study notes integration
- API access
